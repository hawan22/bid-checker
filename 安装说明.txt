========================================
  标书CT扫描仪 v1.7 — 安装说明
========================================

【软件说明】
帮你检查标书的常见错误：
资质有效期 / 签章 / 价格大小写 / 废标关键词
日期矛盾 / 名称不一致 / 页码连续性 / 字体规范
目录完整性 / 页眉页脚 / 联合体协议 / 金额核对...
共 9 个功能模块，一键生成诊断报告。

所有文件在本地处理，不上传网络，关闭即清除记录。


----------------------------------------
【第一步：安装 Python（只需做一次）】
----------------------------------------

1. 打开浏览器，访问：https://www.python.org/downloads/
2. 点击黄色大按钮 "Download Python 3.x.x"
3. 运行下载的安装程序
4. ⚠️ 务必勾选底部的 "Add Python to PATH" ← 很关键！
5. 点 "Install Now"，等待完成


----------------------------------------
【第二步：安装软件并启动（双击一个文件）】
----------------------------------------

解压缩包后，双击：

   👉  一键安装并启动.bat

程序会自动安装所需组件（需联网，首次约 3~8 分钟）
完成后浏览器会自动打开，地址：http://localhost:8501


----------------------------------------
【以后每次启动】
----------------------------------------

直接双击：  run.bat
浏览器打开 http://localhost:8501 即可使用


----------------------------------------
【可选：安装 OCR 图片识别（拍照上传用）】
----------------------------------------

如需识别手机拍照的图片文字，请额外安装 Tesseract：

1. 访问：https://github.com/UB-Mannheim/tesseract/wiki
2. 下载 tesseract-ocr-w64-setup-x.x.x.exe（选最新版）
3. 安装时展开语言列表，勾选 "Chinese Simplified"
4. 安装完成，重启软件即可

（不安装 Tesseract 也可正常使用，只是无法识别图片中的文字）


----------------------------------------
【常见问题】
----------------------------------------

Q: 双击 bat 文件闪一下就关了？
A: 检查 Python 是否安装时勾选了 "Add to PATH"，重新安装 Python 再试

Q: 浏览器没自动打开？
A: 手动打开浏览器，输入 http://localhost:8501

Q: 端口被占用？
A: 修改 run.bat 里的 8501 改成其他数字（如 8502）
