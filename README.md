对标书做全身检查，找出常见错误和不规范项。**所有处理在本地完成，文件不上传任何服务器，关闭后无任何留底。**

| Tab | 功能 |

## 🚀 安装与使用

### ✅ 方式一：发给任何人用（推荐）

**在本机打包：**
1. 双击 `打包发送.bat`
2. 生成 `标书CT扫描仪_portable_XXXXXXXX.zip`
3. 通过微信/U盘/百度网盘发给对方

**对方收到后（首次，约5分钟）：**
1. 解压到任意文件夹
2. 安装 [Python 3.10+](https://www.python.org/downloads/)（安装时**务必勾选 "Add Python to PATH"**）
3. 双击 `一键安装并启动.bat`，自动安装依赖，完成后启动
4. 以后每次只需双击 `run.bat`

### ✅ 方式二：本机使用

双击 `run.bat` 即可，首次会自动安装依赖。

### ✅ 方式三：局域网共享（同一WiFi）

双击 `run_share.bat`，黑窗口会显示局域网地址，同事/手机扫码即可访问。

## ⚠️ 注意事项

## 📦 功能模块

| 模块 | 功能 |
|------|------|
| 📄 PDF可复制性 | 检测PDF是否为文字版，还是不可识别的扫描图片版 |
| 📅 资质有效期 | 自动提取证件有效期，标注过期/即将过期（红黄绿） |
| 📊 技术参数比对 | 上传招标参数表 + 投标响应表，自动逐行对比高亮差异 |
| 💰 价格大小写 | 将数字金额自动转换为中文大写，验证一致性 |
| 🔏 签章检测 | 检测PDF是否含合规数字签名（区分图片印章vs数字签章） |

## 🚀 快速启动（任意电脑）

### 方式一：双击运行（Windows）
1. 确保已安装 [Python 3.10+](https://www.python.org/downloads/)（安装时勾选"Add to PATH"）
2. 双击 `run.bat`，浏览器自动打开

### 方式二：命令行
```bash
pip install -r requirements.txt
streamlit run app.py
```
浏览器访问 `http://localhost:8501`

### 方式三：免安装（Streamlit Cloud）
1. 把本项目上传到GitHub（私有仓库即可）
2. 访问 [share.streamlit.io](https://share.streamlit.io) 连接仓库一键部署
3. 获得一个网址，任何浏览器都能直接访问，无需安装任何软件
